<a name="鹿功能说明"></a>

## 🦌 鹿功能说明

### 基础签到

| 命令                   | 说明            |
| -------------------- | ------------- |
| `🦌` 或 `鹿`           | 进行今日签到（有失败概率） |
| `补🦌[日期]` 或 `补鹿[日期]` | 补签指定日期        |
| `戒🦌[日期]` 或 `戒鹿[日期]` | 减少指定日期的签到次数   |
| `看🦌` 或 `看鹿`         | 查看自己的签到面板     |
| `看🦌 @某人`            | 查看他人的签到面板     |

### 🎯 倍数签到

| 命令                 | 说明               |
| ------------------ | ---------------- |
| `🦌×n` 或 `鹿×n`     | 消耗 n 个杯面，尝试倍数签到  |
| `解除鹿失败` 或 `解除🦌失败` | 消耗 10 杯面解除当日失败锁定 |

**注意**：倍数签到有失败风险，失败仅扣除杯面且当天无法再签到。

### 🤝 鹿友系统

| 命令            | 说明                    |
| ------------- | --------------------- |
| `添加鹿友 @某人`    | 添加鹿友（对方才能帮你签到）        |
| `添加鹿友 all`    | 开启全员鹿友模式，所有人都可以帮你签到   |
| `绝交鹿友 @某人`    | 删除鹿友                  |
| `绝交鹿友 all`    | 关闭全员鹿友模式              |
| `我的鹿友`        | 查看我的鹿友列表及全员模式状态       |
| `帮鹿 @某人`      | 帮鹿友签到一次               |
| `帮鹿×n @某人`    | 消耗 n 杯面帮鹿友倍数签到        |

**注意**：
- 只有被添加为鹿友的人才能帮你签到
- 开启全员鹿友模式后，所有人都可以帮你签到，无需单独添加

### 🏆 排行榜

| 命令             | 说明              |
| -------------- | --------------- |
| `🦌榜` 或 `鹿榜`   | 查看本群签到次数排行榜（降序） |
| `戒🦌榜` 或 `戒鹿榜` | 查看本群签到次数排行榜（升序） |

排行榜每月重置一次。

<a name="杯面返还机制"></a>

## 💰 杯面返还机制

当鹿成功完成指定任务时，系统会按照预设倍数自动返还相应数量的杯面。

### 返还规则

| 条件       | 返还倍数 | 说明           |
| -------- | ---- | ------------ |
| 普通成功     | 1倍   | 返还与投入数量相等的杯面 |
| `鹿×3` 成功 | 3倍   | 返还投入数量3倍的杯面  |

### 示例

- 发送 `鹿×3` 成功：投入3杯面 → 返还9杯面（3×3）
- 发送 `鹿×5` 成功：投入5杯面 → 返还5杯面（5×1）
- 发送 `帮鹿×3 @某人` 成功：投入3杯面 → 返还9杯面（3×3）

### 返还管理命令

| 命令       | 说明                 |
| -------- | ------------------ |
| `我的返还日志` | 查看最近10条返还记录        |
| `我的返还统计` | 查看返还统计信息（总次数、总收益等） |

<a name="kkb互动系统"></a>

## 💕 KKB互动系统

KKB（扣扣B）是一个趣味互动系统，消耗杯面对群友进行"注入生命因子"操作。

> 💡 **提示**：所有 `#kkb` 指令均可使用 `#ccb` 作为别名，例如 `#ccb @目标`、`#ccbinfo` 等。

### 基础操作

| 命令               | 说明                  |
| ---------------- | ------------------- |
| `#kkb @目标` / `#ccb @目标` | 对目标进行 扣扣B 操作（需消耗杯面） |
| `#kkb` / `#ccb`       | 随机选择一位幸运群友进行 扣扣B   |
| `#kkbinfo [@目标]` / `#ccbinfo [@目标]` | 查询 扣扣B 信息（不填则查询自己）  |

### 保护模式

| 命令               | 说明             |
| ---------------- | -------------- |
| `#kkb --on` / `#ccb --on`      | 关闭保护模式，允许被 扣扣B |
| `#kkb --off` / `#ccb --off`     | 开启保护模式，禁止被 扣扣B |
| `#kkb --on @用户` / `#ccb --on @用户`  | 允许特定用户对你 扣扣B   |
| `#kkb --off @用户` / `#ccb --off @用户` | 禁止特定用户对你 KKB   |

### 排行榜

| 命令          | 说明             |
| ----------- | -------------- |
| `#kkbtop` / `#ccbtop`   | 被次数排行榜 TOP5    |
| `#kkbvol` / `#ccbvol`   | 被注入量排行榜 TOP5   |
| `#kkbmax` / `#ccbmax`   | 单次最大注入排行榜 TOP5 |
| `#kkbcharm` / `#ccbcharm` | 魅力榜 TOP5       |

### 游戏机制

- **消耗杯面**：每次 KKB 需要消耗一定数量的杯面
- **炸膛风险**：有概率触发炸膛，需要支付修理费才能继续
- **暴击机制**：有概率触发暴击，注入量翻倍
- **冷却时间**：切换保护状态有冷却时间限制
