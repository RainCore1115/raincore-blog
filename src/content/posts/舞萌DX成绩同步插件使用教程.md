# 舞萌DX成绩同步插件使用教程

## 简介

这是一个基于好友VS功能的舞萌DX成绩同步插件，可以自动从官网抓取成绩并同步到水鱼和落雪查分器。通过这个插件，您可以轻松管理自己的游戏成绩数据。

## 快速开始

### 第一步：绑定好友代码

1. 在舞萌DX公众号中查看您的好友代码
   - 打开微信 → 舞萌DX公众号 → 好友 → 查看好友号码
2. 发送绑定命令
   ```
   #knd bind <好友代码>
   ```
   例如：`#knd bind 1234567890123`
3. 等待系统处理，如果需要添加好友，系统会提示您同意好友申请

### 第二步：绑定查分器

#### 水鱼查分器绑定

**方法一：使用Token绑定**

1. 用手机浏览器打开 <https://maimai.diving-fish.com/>
2. 点击登录并同步数据（如果没有账号请先注册）
3. 登录后点击"编辑个人资料"
4. 填入昵称和QQ号
5. 取消勾选"非网页查询的成绩使用掩码"
6. 在"成绩导入token"处，按照提示点击按钮复制token
7. 发送绑定命令：
   ```
   #knd 绑定水鱼 <刚才复制的token>
   ```

**方法二：使用账号密码绑定**

```
#knd 绑定水鱼账号 <用户名> <密码>
```

#### 落雪查分器绑定

1. 用手机浏览器打开 <https://maimai.lxns.net/>
2. 点击登录（如果没有账号请先注册）
3. 登录后点击左上角三条杠
4. 点击"账号详细" → "舞萌DX" → "第三方应用"
5. 下滑找到"个人API密钥"，点击复制按钮
6. 发送绑定命令：
   ```
   #knd 绑定落雪 <刚才复制的API密钥>
   ```

## 日常使用

### 同步成绩

发送以下命令即可自动同步成绩并导出到已绑定的平台：

```
导
```

或

```
ccb
```

系统会自动完成以下操作：

1. 从舞萌DX官网抓取最新成绩
2. 自动导出到所有已绑定的查分器平台

### 查看同步进度

在同步过程中，您可以随时发送：

```
#knd 状态
```

查看当前同步任务的进度。

### 其他常用命令

- `#knd sync` - 仅同步成绩，不自动导出
- `#knd info` - 查看个人信息和绑定状态
- `#knd 更新水鱼` - 手动导出到水鱼查分器
- `#knd 更新落雪` - 手动导出到落雪查分器
- `#knd 更新全部` - 同时导出到水鱼和落雪

## 高级功能

### 自动导出

开启后，每次同步完成会自动导出到指定平台：

```
#knd 自动导出水鱼开
#knd 自动导出落雪开
```

关闭自动导出：

```
#knd 自动导出水鱼关
#knd 自动导出落雪关
```

### 闲时更新

开启闲时更新后，Bot会在凌晨高成功率时段自动更新您的成绩：

```
#knd 闲时更新开
```

查看闲时更新状态：

```
#knd 闲时状态
```

## 常见问题

### Q: 同步失败怎么办？

A: 请检查以下几点：

1. 确保已在舞萌DX公众号中同意好友申请
2. 检查网络连接是否正常
3. 使用 `#knd 状态` 查看具体错误信息
4. 是否绑定了水鱼或者落雪

<br />

### Q: 支持哪些查分器？

A: 目前支持：

- 水鱼查分器 (maimai.diving-fish.com)
- 落雪查分器 (maimai.lxns.net)

### Q: 如何解绑？

A: 发送 `#knd 解绑` 即可解除好友代码绑定。
